SimpleSave

SimpleSave is a super simple web server that allows for easy storage and retrieval of key-value pairs of strings.

View on GitHub Download .zip Download .tar.gz

Welcome to Simplicity.

SimpleSave is a super simple web server written in python.
It is used for storing key-value pairs on strings. Storing a key-value pair:

   wget http://localhost/?k=key&v=value

Retrieving a previously stored value:

   wget http://localhost/?k=key
